## Déploiement sur GitHub Pages

1. Crée un repo nommé `ton-username.github.io`
2. Pousse tout le dossier `portfolio/` à la racine du repo
3. Ton site sera en ligne sur `https://ton-username.github.io`

## Ce qu'il faut ajouter

1. **Ta photo** → `images/photo.jpg`
2. **Screenshots projets pro** → dans `images/projects/`
3. **Screenshots projets perso** → dans `images/personal/`
4. **Ton CV** → `documents/cv-bachir-mahamane-iro.pdf`
5. **Tes vrais liens GitHub/LinkedIn** → dans le footer du `index.html`
6. **Les liens GitHub de chaque projet** → remplacer les `href="#"` dans le HTML
